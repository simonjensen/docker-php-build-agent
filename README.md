# php-build-agent

Docker image containing the most common tools for building a PHP application.

Comes with the following tools:

- curl
- wget
- jq
- python 3.6 & pip
- aws-cli
- docker-ce
- nodejs 8
- Bower, Gulp & node-sass
- Composer
- php 7.4
  - php7.4-bcmath
  - php7.4-curl
  - php7.4-gd
  - php7.4-intl
  - php7.4-json
  - php7.4-mbstring
  - php7.4-mysql
  - php7.4-odbc
  - php7.4-opcache
  - php7.4-pgsql
  - php7.4-soap
  - php7.4-xml
  - php7.4-zip
  - php7.4-ldap
