# php-build-agent

Docker image containing the most common tools for building a PHP application.

Comes with the following tools:

- curl
- wget
- jq
- python 2.7 & pip
- aws-cli
- docker-ce
- nodejs 8
- Bower, Gulp & node-sass
- Composer
- php 7.2
  - php7.2-bcmath
  - php7.2-curl
  - php7.2-gd
  - php7.2-intl
  - php7.2-json
  - php7.2-mbstring
  - php7.2-mysql
  - php7.2-odbc
  - php7.2-opcache
  - php7.2-pgsql
  - php7.2-soap
  - php7.2-xml
  - php7.2-zip
